Three parallel research agents investigated the open technology questions for the v2-embodiment change: (1) redistributable head-mesh sourcing and licensing (MakeHuman/MPFB2, Blender Studio base meshes, Smithsonian 3D, VRoid, Ready Player Me, scan repositories); (2) practical routes for authoring the 27 canonical rig morphs (glTF-Transform remap, Blender donor transfer, procedural fallback, open-source auto-generators) including ARKit-52 and Oculus-15 mapping tables; (3) cloud TTS viseme metadata formats (Azure Speech viseme IDs and 55-blendshape frames, Amazon Polly viseme speech marks, Google Cloud TTS timepoints, ElevenLabs character alignment) with fixture-provenance and ToS analysis. Every claim carries a primary-source link (licence texts, official API docs, repo READMEs). Findings are distilled into res.asset-sourcing, res.morph-authoring, and res.viseme-provider-format.
